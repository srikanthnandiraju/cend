How it works

CEND.IN  makes it easy for you to send animated cards in your favourite chat message app.
To send a card simply type a url in the following format
http://cend.in/occasion/to/message/from

For example, say if Spiderman wants to send a card to Super man on his birthday, he would type a url like this in his whatsapp chat
http://cend.in/birthday/superman/Happy-returns-of-the-day/Spiderman

Note make sure that you type a - inplace of a blank space to make the URL clickable.

Start using the Card creation app to preview and send cards to your chat apps.
